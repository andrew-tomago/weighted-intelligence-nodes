# Weighted Intelligence Nodes MVP Specification

Last Updated: 2026-02-06
Version: 0.2.0

## Product Goal

Build a lightweight, repeatable workflow that:

1. profiles audience targets from public signals,
2. evaluates candidate content through a committee of persona experts,
3. returns a balanced synthesis that highlights consensus and dissent.

## Non-Goals (Hackathon Scope)

* Full autonomous web scraping stack.

* PII enrichment or private data integration.

* Production-grade model orchestration or infra.

## Data Contracts

| Contract         | Required Fields                                      |
| ---------------- | ---------------------------------------------------- |
| `targets.json`   | `targets[].id`, `targets[].public_data`              |
| `content.json`   | `content[].id`, `content[].title`, `content[].body`  |
| `committee.json` | `experts[].id`, `experts[].weight`, `rubric_weights` |

Optional in `committee.json`: `judging_criteria[]` with `id`, `label`, `weight`, `mapped_metric`, `description`.

## Pipeline Stages

| Stage      | CLI Command                                       | Output                  |
| ---------- | ------------------------------------------------- | ----------------------- |
| Profile    | `python3 scripts/mvp_committee.py profile ...`    | `profiles.json`         |
| Evaluate   | `python3 scripts/mvp_committee.py evaluate ...`   | `committee_matrix.json` |
| Synthesize | `python3 scripts/mvp_committee.py synthesize ...` | `summary.md`            |
| End-to-end | `python3 scripts/mvp_committee.py run ...`        | all outputs             |

## Scoring Model (MVP v0.2)

| Dimension | Intent                   | Signal                                              |
| --------- | ------------------------ | --------------------------------------------------- |
| Fit       | Audience-topic alignment | Normalized keyword overlap vs profile interests     |
| Clarity   | Readability              | Sentence-length heuristic                           |
| Novelty   | Freshness                | Unique-token ratio                                  |
| Trust     | Credibility              | Numeric + source-term density                       |

Overall score: persona-adjusted metric weighting + focus alignment bonus.

Judge feedback now synthesizes:
- judging criteria mapped to metrics (`criteria_scores`)
- persona lens emphasis (`lens_metric_weights`)
- empathetic rationale text (`rationale`)

## Acceptance Criteria

* Running `run` creates all output files without manual edits.

* Committee output includes per-expert opinions, rationale, and aggregate synthesis.

* Summary clearly identifies strongest pro and strongest con.
